# Getting Started

## Installation

```shell
npm i nes-vue
```
## Import Component

```vue
<template>
  <nes-vue
    url="https://taiyuuki.github.io/nes-vue/Super Mario Bros (JU).nes"
  />
</template>
<script setup>
  import { NesVue } from 'nes-vue';
</script>
```
